# コーディング規約

## JavaScript

ファイル名はハイフン区切り、小文字のみ使用。拡張子はjs。

改行コードはLF。

クラス名はUpperCamelCase。

定数、変数名、メソッド名、関数名はlowerCamelCase。

## PHP

クラス名はUpperCamelCase。

メソッド名、関数名、変数名はlowerCamelCase。
